## Cách cài đặt và chạy project

### Clone project
    git clone https://github.com/nqt27/Position-Size-Calculator.git
    cd Position-Size-Calculator
### Cài đặt package PHP
    composer install
### Tạo file môi trường
    copy .env.example .env
### Chạy migration
    php artisan migrate
### Chạy server
    php artisan serve
    Mở trình duyệt tại:
    http://127.0.0.1:8000

##  Laravel version

- **PHP:** >= 8.2  
- **Laravel:** 12.x
